## 주요 기능
> ## 일반
> * 사용자(고객/관리자) 로그인/회원가입
> * JWT 토큰, 인증/인가
> * 권한에 따른 페이지 접근 처리

> ## 고객 사용자
> * 상품 조회
> * 장바구니에 상품 담기/수정/삭제
> * 장바구니 상품 결제
> * 주문 배송 상태 확인

> ## 관리자 사용자
> * 상품 추가/수정/삭제
> * 주문 내역 관리
